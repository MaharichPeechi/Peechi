# Português [0.1.1] Correção da concatenação de texto

## Objetivo

Corrigir a regra do operador `+` para não fazer conversão automática de tipos.

## Regra oficial nesta versão

```text
número + número → soma
texto + texto   → concatenação
texto + número  → erro
número + texto  → erro
outros casos    → erro
```

## Exemplos válidos

```pee
print(10 + 5)
print("Olá, " + "mundo!")
print("Idade:", 15)
```

## Exemplos que devem dar erro

```pee
print("Texto" + 1)
print(1 + "Texto")
```

# English [0.1.1] Text Concatenation Correction

## Objective

Correct the `+` operator rule to prevent automatic type conversion.

## Official Rule in this Version

```text
number + number → sum
text + text → concatenation
text + number → error
number + text → error
other cases → error

```

## Valid Examples

```pee
print(10 + 5)
print("Hello, " + "world!")
print("Age:", 15)

```

## Examples that should give an error

```pee
print("Text" + 1)
print(1 + "Text")
```
