# Português [0.1.0] Núcleo inicial

## Implementado

- `print(...)`
- strings com aspas duplas e simples
- números
- booleanos
- variáveis por atribuição direta
- `+`, `-`, `*`, `/`
- `==`, `!=`, `>`, `<`, `>=`, `<=`
- `if`
- `else`
- `while`
- `loop`
- `break`
- comentário com `//`
- concatenação de texto com `+`, somente `texto + texto`
- múltiplos argumentos em `print(...)`

## Observações

- `print` separa múltiplos argumentos com espaço.
- Blocos ainda não criam escopo próprio nesta fase.
- `texto + número` e `número + texto` devem dar erro nesta fase.
- `continue`, `null`, `and`, `or` e `not` são blocos futuros.

# English [0.1.0] Initial Core

## Implemented

- `print(...)`
- Strings with double and single quotes
- Numbers
- Booleans
- Variables by direct assignment
- `+`, `-`, `*`, `/`
- `==`, `!=`, `>`, `<`, `>=`, `<=`
- `if`
- `else`
- `while`
- `loop`
- `break`
- Comment with `//`
- Text concatenation with `+`, only `text + text`
- Multiple arguments in `print(...)`

## Notes

- `print` separates multiple arguments with a space.
- Blocks do not yet create their own scope at this stage.
- `text + number` and `number + text` should give an error at this stage.
- `continue`, `null`, `and`, `or`, and `not` are future blocks.
